# 分析-Postviewer

**题目标题：**Postviewer v5² (FF)


**题目描述：**
One Postviewer but two flags. Can you get the flag?
Should be easier than the other version! :)

Note: This challenge has a static scoring.


题目靶机网站：http://localhost:1337/


## 要求

1. **绝对不能进入docker获取源码，只能黑盒分析**，做出题目。因为这道题的互联网版本就是黑盒，所以黑盒分析才是最真实的环境。
2. 不能作弊！作弊指的是：根据题目去搜索或查阅现成的解法。我希望你能根据题目内容，自己分析出结果，不是不允许你搜索而是不能直接查看别人已经做好的解题文档（这样做就是作弊）。
